# Features

Bu klasör LexiCode projesinin tüm kaynak kodlarını içerir:

- `app.py` → Flask uygulaması ve API rotaları
- `desktop_launcher.py` → masaüstü başlatıcı
- `templates/` ve `static/` → arayüz dosyaları
- `services/` → iş mantığı servisleri
- `utils/` → yardımcı fonksiyonlar
- `tests/` → otomatik testler
